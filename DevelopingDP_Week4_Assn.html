Introduction

This data set contains statistics, in arrests per 100,000 residents for assault, murder, and rape in each of the 50 US states in 1973. Also given is the percent of the population living in urban areas. A data frame with 50 observations on 4 variables.

[,1] Murder numeric Murder arrests (per 100,000) [,2] Assault numeric Assault arrests (per 100,000) [,3] UrbanPop numeric Percent urban population [,4] Rape numeric Rape arrests (per 100,000)

Shiny Web Application

  • The Shiny Web Application developed by using USArrests Dataset from inbuilt R package.
  • The developed shiny App has 2 parts, a drop down box on left panel and main panel is on the right
  • The Crime drop down box facilitates the user to choose 3 different crime rates in USA 1973
  • Based on the selection in the drop down menu, it is possible to see the associated data,map,regression diagnostics on the tabs in the main panel.
